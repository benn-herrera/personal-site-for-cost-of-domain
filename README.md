# personal-site-for-cost-of-domain
This repo has a guide to take you from zero to launch of a personal site, plus the sources for a Cloudflare Worker project to back it.
The Worker project has tooling to support a Markdown authoring -> Web 1.0 HTML deployment workflow.

If you want to know why, see [deshittification↗](https://bennherrera.dev/writing/deshittification/) at [bennherrera.dev](https://bennherrera.dev)

To get get set up start with [ZERO_TO_LAUNCH.md](ZERO_TO_LAUNCH.md)

[AUTHORING_README.md](AUTHORING_README.md) explains the features, authoring process, and tooling.
